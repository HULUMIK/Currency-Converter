## 📖 Описание

«Currency Converter» — графическое приложение для конвертации валют с использованием актуальных курсов из внешнего API ([ExchangeRate-API](https://exchangerate-api.com/)). Программа сохраняет историю всех конвертаций в JSON-файл и позволяет просматривать её в удобной таблице.

## ✨ Возможности

- **Конвертация валют** — выбор из 30+ валют
- **Актуальные курсы** — автоматическая загрузка из API
- **История операций** — сохранение всех конвертаций с датой, курсом и результатом
- **Валидация ввода** — сумма должна быть положительным числом
- **Автономный режим** — при недоступности API используются стандартные курсы
- **Очистка истории** — кнопка для удаления всех записей
